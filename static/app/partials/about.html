AngularJS + Cloud Endpoint

Sample application for using AngularJS backed with a Cloud Endpoint on the Google App Engine. This is a single-page application that loads resources and data as needed based on URL routing.

Client

This part of the application runs in your browser.

  • AngularJS 1.3.15, HTML enhanced for web apps.
    • angular-resource, support for interacting with RESTful server-side data sources.
    • ui-router 0.2.14, state based and nested routing from the Angular UI package.
    • angular-scenario, E2E testing. Simulate user interaction with the client application.
  • Boostrap 3.3.4, front end framework for faster and easier web development.
  • Moment.js 2.10.2, date and time parsing and formatting library.
  • Jasmine 1.3.1, behavior driven test framework. Uses angular-mocks to handle testing of AngularJS modules.

App Engine

This part of the application runs and stores data on Google's scalable systems.

  • Python SDK 1.9.19, application platform.
    • Python NDB, schemaless object datastore.
    • Cloud Endpoints, easily create a web service API.
    • Endpoints Proto Datastore, allows an NDB model to be used directly with a Cloud Endpoints API.

Test

One of the great things about this application platform is that it includes excellent testing tools.

  • Unit Test, tests for the individual Angular application components.
  • E2E Test, uses the Angular Scenario Runner to simulate user interaction with the application.
  • API Explorer, browse the methods available on our Cloud Endpoint. Send commands and view results.
